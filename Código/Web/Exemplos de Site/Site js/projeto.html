Objetivos

Criamos este site com o objetivo de divulgar o Reggae, temos como público alvo todas as pessoas que gostam de música, não só os que curtem este estulo em questão.

Criaremos uma página onde responderemos as questões enviadas pelos internautas.

Nosso site não possui restrições, todos os que gostam de música estão convidados a nos deixar um comentário ou pergunta.

O único artifício utilizado para a produção desse site foi a implementação em HTML.

Nossa maior dificuldade foi devido a falta de experiência em criar sites e o tempo para nos adaptar a linguagem.
